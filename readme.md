# 🚀 RecruitAI Pro — Visual Resume Screening

An advanced **AI-powered Resume Screening Web App** that helps recruiters analyze, score, and shortlist candidates based on job descriptions.

Built with a futuristic UI and intelligent processing, this tool simulates real-world hiring workflows with automation and analytics. 

---

## ✨ Features

* 📄 Upload resumes (images or text)
* 🧠 AI-powered candidate analysis (Anthropic API)
* 📊 Smart scoring system (0–100)
* ✅ Automatic shortlist / reject decision
* 🔍 Skill matching & missing skills detection
* 📈 Real-time progress tracking
* 📋 Analysis history with saved sessions
* 🎯 Filter candidates (All / Shortlisted / Rejected)
* ⚡ Modern, premium UI with animations

---

## 🧠 How It Works

1. Enter a **Job Title + Description**
2. Upload candidate resumes (image or text)
3. Click **Analyze & Shortlist**
4. AI processes each resume:

   * Extracts candidate details
   * Matches skills with job requirements
   * Calculates a score
   * Generates a hiring verdict
